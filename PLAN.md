## MiCoach

A tutor capable of analyzing CVs and Github repos that can create roadmaps to improve skills and reduce gaps between current user skills with the desired role featured with chatbot and code training exercises. Tutor must be capable of:

1. Must analyse users skills set based on pdf CV and public GitHub repos.
2. Must create a roadmap based on the analysis for the career path he had chosen in the registration.
    2.1 Career path planning must be based on units
    2.2 Must provide projects for user practice
3. Must create a test for each unit final test
    3.1 User must provide a public GitHub repo in order to be graded and evaluated with feedback
    3.2 User cannot pass if he does not aprooves previews units
    3.3 User cannot use the chatbot as solver, this is prohibitted
4. User can interact with the tutor/couch at any moment in a chat bot that can resolve doubts
5. User can practice with AI created exercises based on created roadmap
    5.1 User may change difficulty (easy, medium and hard)
    5.1 After submiting the problem the user may recieve feedback from its solution, passing or not the testing part
6. User may mock interviews at any moment based on chosen career path
    6.1 Questions have different difficulty and abstraction
    6.2 Feedback will tell the user how it went with details
